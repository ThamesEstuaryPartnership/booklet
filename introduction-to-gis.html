Introduction to GIS

GIS is the abbreviation for geographic information systems. It is a computer system used to create, manage, visualise, analyse and interpret spatially referenced data.

GIS helps answer questions such as What is on top of what?, What is next to what?, What is connected to what?, How has it changed over time and space? and make decisions.

Geographic information

Geographic information is made up of spatial location data and attribute data.

  • Spatial location data answers the question where something is.

  • Attribute data answers the question what something is.

GIS capabilities

GIS has many capabilities. These are:

  • Spatial analysis: different tools are used to identify trends, patterns and relationships in data.

  • Map making: to spatially represent data from multiple sources (e.g. spreadsheets). This is the most common use of GIS.

  • Imagery and remote sensing: to extract information from imagery and remotely sensed data (e.g. satellites).

  • Data collection and management: to access, use, integrate and store data in a GIS.

GIS data

Types

Vector data

Spatial data that are stored as geometrical shapes such as:

  • Points: Points are single x,y coordinate locations.

  • Lines: Lines are defined by two or more points that are connected with lines.

  • Polygons: Polygons are defined by multiple points that are connected with lines and are closed.

Vector data represents discrete data such as buildings, rivers or ponds.

Vector types and vector data drawn on a map


Raster data

Raster data is made up of a matrix of pixels, called cells, with each containing a value that represents the conditions for the area covered by that cell. Raster data represents continuous data (integer or real values) such as temperature or elevation.
